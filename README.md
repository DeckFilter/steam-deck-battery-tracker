# Steam Deck Battery Tracker

Simple plugin to track battery history for your Steam Deck as well as power stats.

This project is forked from [safijari/steam-deck-battery-tracker](https://github.com/safijari/steam-deck-battery-tracker) to provide support for more handhelds and to keep the plugin maintained. Huge thanks to the original author.

Tested devices:

- Legion Go S
- Legion Go
- Steam Deck
- ROG Ally
- Zotac Zone
